#1-10. HTML이 중요한 이유

의미에 맞는 태그로 웹페이지를 만드는 지식인
정보사회에서 HTML을 의미에 맞게 잘 사용하는 것은 비즈니스적인 측면에서 중요하다.

웹의 핵심적인 철학은 접근성(Accessibility)
HTML을 의미론적으로 잘 사용하는 것이 중요하다.
HTML은 비즈니스+휴머니즘적인 측면에서 중요한 기술

#1-12. 부모 자식과 목록

#1-13. 문서의 구조와 슈퍼스타들

#1-16. 원시웹

수업의 목표
1) 웹페이지를 만드는 것
2) 만든 웹페이지를 인터넷을 통해 누구나 가져갈 수 있도록 하는 것

웹의 역사
Internet VS WEB

팀 버너스 리는 웹의 창시자
세계 최초의 웹브라우저인 월드 와이드 웹(world wide web)을 만든 사람
훗날 웹 브라우저의 이름인 WWW가 웹의 이름이 된다.
이 서버에 info.cern.ch라는 도메인 네임을 부여 -> 세계 최초의 웹페이지

WEB1은 원시웹을 다루는 수업


#1-17. 인터넷을 여는 열쇠: 서버와 클라이언트

2대의 컴퓨터 (인터넷으로 연결됨)
1) Web Browser - http://info.cern.ch/index.html 접속 -> 요청(고객, 클라이언트client)
2) Web Server - index.html 소지, http://info.cern.ch -> 응답 (사업자, 서버server)

요청하는 컴퓨터는 클라이언트 컴퓨터
응답하는 컴퓨터는 서버 컴퓨터

웹브라우저는 클라이언트 컴퓨터에서 동작, 그래서 웹브라우저는 다른 말로 웹 클라이언트라고 한다.
웹서버는 서버 컴퓨터에서 동작, 그래서 웹서버라고 한다.
